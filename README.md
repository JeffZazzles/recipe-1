# Roasted Cherry Tomato Chicken Salad

A shareable single-page recipe site for roasted cherry tomato chicken salad with creamy Dijon yogurt dressing.

## Publish with GitHub Pages

1. Create a new GitHub repository, for example `recipe-page`.
2. Upload `index.html` to the repository root.
3. In GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select your default branch, usually `main`, and the root folder `/`.
6. Save. GitHub will publish the page once Pages finishes building.
